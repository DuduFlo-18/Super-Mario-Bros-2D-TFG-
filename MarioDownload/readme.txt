Proyecto de Fin de Grado - Desarrollo de Aplicaciones Multiplataforma (DAM)

Este proyecto consiste en la creación de un videojuego 2D inspirado en el clásico Super Mario Bros, desarrollado con el motor de videojuegos Unity. 
El objetivo principal ha sido aplicar todos los conocimientos adquiridos a lo largo del ciclo formativo, integrando programación, diseño de niveles, 
gestión de estados, animaciones y publicación multiplataforma (Android y Windows).

Características del Juego

- Control completo del personaje (movimiento, salto, disparo, power-ups)
- Transformaciones: Mario pequeño, grande y de fuego
- Ciclo completo de niveles con checkpoints, HUD y sistema de vidas
- Sistema de puntuación y monedas con 1UPs
- IA básica de enemigos y jefe final (Bowser)
- Sistema de sonido y música (incluyendo estrella invencible)
- Soporte para controles táctiles en móviles
- Guardado de progreso con PlayerPrefs
- Estructura modular (GameManager, LevelManager, InputTranslator...)


Tecnologías y Herramientas

- Motor: Unity 2021.3.26f1 (LTS)
- Lenguaje: C# (.NET Standard 2.1)
- Plataformas: Android & Windows (Teclado/Mando)
- Control de versiones: Git & GitHub
- IDE:** Visual Studio Code
- Assets:** Sprites y sonidos libres/propios
